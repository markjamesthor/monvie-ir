Problem

개인화 동화책은 전부 수동이다

부모가 사진을 보내면 디자이너가 포토샵으로 편집한다. 1권에 3일, 인건비가 원가의 대부분. 가격은 8~15만원. 스케일이 안 된다.

Solution

AI가 실시간으로 만든다

주문부터 인쇄 발주까지 사람 개입 0인 자동화 파이프라인.

사진 업로드
↓ AI 배경 제거 (6종 모델 자동 선택) 0.5초
↓ 포즈 분석 + 스마트 크롭 0.3초
↓ 품질 자동 검수 (8개 항목) 0.2초
↓ 동화책 자동 생성 + 모바일 미리보기 즉시
↓ 부모 승인 → 인쇄 파일 생성 → 자동 발주

Moat

왜 따라잡기 어려운가

  • 01
    정신의학 기반 제품 설계
    창업자가 소아정신의학 전공. 매 페이지가 부모-아이 상호작용을 유도하도록 설계되어 있다. "이름이 들어간 책"이 아니라 치료적 의미가 있는 책.
  • 02
    7년, 수천 권의 수동 제작 경험
    어떤 사진이 인쇄에서 깨지는지, 어떤 포즈가 장면에 맞는지, 어디서 클레임이 나는지. 수천 장의 아이 사진을 직접 다뤄본 사람만 아는 것들.
  • 03
    AI 모델 6종 자체 서빙
    외부 API 의존 없음. 사진 특성별 최적 모델 자동 선택. 자체 GPU에서 처리하므로 아이 사진이 외부로 나가지 않는다.
  • 04
    Config-Driven 테마 시스템
    새 동화책 = JSON 파일 1개. 코드 수정 없이 테마 무한 확장. 이미 6개 테마 보유.
